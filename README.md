# Antigravity Event LP

This is the landing page for the Antigravity events, designed with a SpaceX-inspired "Gravity-Free" aesthetic.

## Deployment

This project is ready for GitHub Pages.
1. Push this repository to GitHub.
2. Go to Settings > Pages.
3. Select `main` branch and `/` (root) folder.
4. Save.

## Development

- `index.html`: Main content.
- `style.css`: Styles (includes Glitch effects and animations).
- `main.js`: Scroll reveal and particle effects.
- `img/`: Assets.

## Features

- **Responsive Design**: Mobile-ready.
- **Vibe Coding**: Built with AI assistance.
- **Immersive Effects**: Parallax, Glitch, Particles.
